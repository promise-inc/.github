## Sobre

A **Promise Fulfilled** é uma software house fullstack especializada em criar produtos digitais de ponta a ponta 
do backend à interface, do deploy ao monitoramento.

Trabalhamos com times enxutos e eficientes, priorizando qualidade de código, entregas consistentes e comunicação direta.

## Stack Principal

| Camada | Tecnologias |
|--------|-------------|
| **Frontend** | React, Vite, Next.js, TypeScript, Tailwind CSS |
| **Backend** | NestJS, Node.js, PostgreSQL, Prisma |
| **Infra** | Vercel, AWS, Docker, PM2 |
| **Mobile** | React Native, Expo |
| **Ferramentas** | GitHub Actions, Figma |

## Produtos Internos

- **Promise Hub** — Plataforma de gestão de projetos com tracking de tempo via Claude Code, Kanban, time tracking e integrações com GitHub.

## Princípios

- **Código limpo** — TypeScript strict, zero `any`, testes onde importa
- **Entregas reais** — Software que funciona em produção, não só em demo
- **Automação** — CI/CD, hooks, monitoramento — menos trabalho manual
- **Transparência** — Clientes acompanham progresso em tempo real
